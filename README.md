# Student Performance Analysis

This project analyzes student performance data using Python.

## Features
- Data preprocessing (cleaning, handling missing values)
- Calculation of average marks
- Identification of highest and lowest scoring students
- Summary statistics of class performance
- Identification of students needing academic support

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib

## Files
- DataAnalysis.ipynb
- StudentsPerformance.csv

## Conclusion
This project helps in understanding student performance and identifying students who need improvement.
